# Open Source Notes

This project is released under the MIT License. See [`LICENSE`](../LICENSE).

## What is included

- Next.js app source under `app/` and `components/`
- structured story data under `data/`
- static public assets under `public/`
- production verification scripts under `scripts/`
- deployment and maintenance docs under `docs/`

## What is intentionally excluded

- real API keys and local env files
- dependency folders and build output
- generated current visual screenshots
- temporary downloads, logs, and local editor state

## Responsible use

The game content is fictional entertainment. If you fork or deploy it, keep clear user-facing context that the scenarios are fictional and avoid using generated AI text to impersonate real people.
